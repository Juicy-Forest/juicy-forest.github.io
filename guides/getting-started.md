## Overview

This guide will help you set up your local development environment for Juicy Forest.

## Prerequisites

- Node.js 18+
- MongoDB
- npm or yarn

## Quick Start

```bash
# Clone the repositories
git clone https://github.com/juicy-forest/backend
git clone https://github.com/juicy-forest/client

# Backend setup
cd backend
npm install
npm run dev

# Client setup (in another terminal)
cd client
npm install
npm run dev
```
